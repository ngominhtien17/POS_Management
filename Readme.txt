# Hướng dẫn sử dụng dự án

## Cài đặt thư viện
Sau khi tải mã nguồn về, bạn cần cài đặt các thư viện cần thiết bằng cách chạy lệnh sau trong thư mục dự án:

```bash
npm install
hoặc 
yarn install
```

Sau khi cài đặt xong, bạn có thể chạy ứng dụng bằng cách sử dụng lệnh sau:

```bash
npm start
hoặc
yarn start
```

Sau khi chạy ứng dụng, bạn có thể đăng nhập vào hệ thống với các tài khoản sau:

Tài khoản Admin:
Tên đăng nhập: admin
Mật khẩu: admin

Tài khoản nhân viên:
Tên đăng nhập: phamphuongngoc509
Mật khẩu: 123456
